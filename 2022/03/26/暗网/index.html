第二代洋葱路由器（The Onion Router，Tor）

为了躲避追踪，利用第二代洋葱路由器（The Onion Router，Tor）实现对暗网的访问：特殊的软件

暗网监视分析的困难

  • 访问暗网的速度受到限制。访问暗网需要特殊软件 Tor，而Tor 的连接过程需要进行加密和解密
  • 隐藏服务地址难以收集
  • 暗网具有匿名性：传统方法无法追踪真是ip和身份

隐藏服务地址收集

  • 对Tor隐藏服务过程以及Tor匿名通信过程的原理进行相关分析，利用部署目录服务器的方法收集
  • 通过分析泄露到 DNS 中的隐藏服务请求来发现隐藏服务地址
  • 根据 Tor 匿名网络自身的特点[7]，通过 UGC（User Generated Content）网站数据提取、在搜索引擎中进行关键词搜索和从暗网域名公开列表中获取这三种方式来进行地址收集
  • 分析 I2P 域名收集的难点，分别从 I2P的地址簿文件中直接获取、爬取 I2P 网站内链接、根据 UGC 中的链接进行匹配获取以及在搜索引擎中搜索特定关键字获取
  • 利用 Tor2web ，提出将特殊关键字输入到普通搜索引擎（如谷歌，Google）中来获取隐藏服务地址的方法，但关键词数量被限制

Tor2web 是一种 HTTP 代理软件，可以通过普通的 web 浏览器访问 Tor 隐藏服务
